# **App Name**: CognitoCrowd

## Core Features:

- Landing Page: Landing Page: A polished, informative marketing site for unauthenticated users.
- Authentication: Authentication: Secure user sign-up and login using email/password. Redirects to main app layout upon completion
- Task Dashboard: Dashboard: Displays a card grid of available tasks. Each card represents a task and includes title, points, description and difficulty
- Task Completion: Task Page: Complete simple, gamified tasks of various types (multiple-choice, text input, ranking). Submission triggers points awarded
- Rewards Page: Profile & Rewards: User profile to track points and task history
- Task Management: Admin Panel: Manage tasks via an interface with dynamic forms that handle different task settings.
- AI Assistant: AI Assistant Chat: A tool providing a simple, clean chat interface for conversing with a Genkit-powered AI assistant.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to evoke a sense of intellectual depth and trustworthiness.
- Background color: Very light gray (#F0F0F0) to provide a clean and unobtrusive backdrop, emphasizing the task content.
- Accent color: Vivid Violet (#A020F0) to draw attention to key interactive elements.
- Headline font: 'Space Grotesk' sans-serif for a modern, tech-forward aesthetic in headers.
- Body font: 'Inter' sans-serif for clear, readable text in task descriptions and other body content.
- Use clean, line-based icons that align with a modern aesthetic.
- Responsive layout that adapts to various screen sizes.
- Subtle, unobtrusive animations to provide feedback during interactions, such as task submission or reward redemption.
